# ScholarOS Academic Onboarding

## First 15 minutes

1. Create or open your workspace vault.
2. Add course notes and PDFs into `raw/`.
3. Open the ScholarOS dashboard and assignment board.
4. Run ingestion to generate concept pages and links in `wiki/`.
5. Review contradictions and approve updates before commit.

## Course setup checklist

- Add course code and semester metadata.
- Define assignment due dates.
- Enable flashcard generation for concept-heavy courses.
- Keep wiki links on each assignment for traceability.

## Weekly rhythm

- Monday: ingest new lecture material.
- Mid-week: review due flashcards and update assignment status.
- Friday: run wiki health check for orphans and contradictions.
