# ScholarOS Flashcard Guide

## FSRS review grades

- `Again (1)`: you could not recall the answer.
- `Hard (2)`: partial recall with effort.
- `Good (3)`: correct recall with moderate confidence.
- `Easy (4)`: instant recall.

## Best practices

- Grade before flipping to avoid hindsight bias.
- Keep cards atomic (one concept per card).
- Review daily to maintain retention curves.
- Link flashcards to concept pages for context.

## Troubleshooting

- If due count looks wrong, refresh flashcard list.
- If cards are missing, verify course ID scope.
- If scheduling feels too aggressive, tune FSRS retention target.
