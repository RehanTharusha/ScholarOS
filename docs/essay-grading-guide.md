# ScholarOS Essay Grading Guide

## Input format

Provide:

- Draft title
- Essay text
- Rubric markdown
- Optional source list

## How grading works

1. Rubric parser extracts criteria and weights.
2. Essay grader scores each criterion.
3. Citation verifier checks support against wiki-linked material.
4. Feedback returns criterion-level comments and revision suggestions.

## Getting stronger feedback

- Use explicit rubric criteria with point values.
- Keep claims concrete and sourceable.
- Include source names used in your draft.
- Regrade after each major revision pass.
